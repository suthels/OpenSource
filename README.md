## OpenSource
群智能算法小狂人公众号开源的所有资源

## 安装环境

- 请先安装[Git](https://git-scm.com/)，需要克隆本仓库
- 前端运行环境[Node](https://nodejs.org/en/download/)，演示Node环境为14.21.0版本

### 克隆我的地址
```shell
git clone  git@github.com:suthels/OpenSource.git
```
### 直接下载仓库内容[直接下载]
```shell
https://github.com/suthels/OpenSource.git
```
